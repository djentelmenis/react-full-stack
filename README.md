# React-Full-Stack

Full stack TypeScript web app starter with React, Express and Webpack.

- React
- Express
- Nodemon
- Webpack
- Babel
- Mocha
- ESLint
- Prettier

## Building and running on localhost

Install dependencies:

```sh
npm i
```

Run development mode:

```sh
npm run dev
http://localhost:3000/
```

Run production mode:

```sh
npm run prod
http://localhost:8080/
```

Run tests:

```sh
npm run test
```
