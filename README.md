# empty-project

Full stack web app starter with React, Node.js, Express and Webpack.

## Building and running on localhost

Install dependencies:

```sh
npm i
```

Developement mode:

```sh
npm run dev
```

Production mode:

```sh
npm run prod
```

Run tests:

```sh
npm run test
```
