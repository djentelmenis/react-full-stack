# React-Full-Stack

Full stack web app starter with React, Node.js, Express and Webpack.

- React
- Node
- Express
- Webpack

## Building and running on localhost

Install dependencies:

```sh
npm i
```

Run development mode:

```sh
npm run dev
```

Run production mode:

```sh
npm run prod
```

Run tests:

```sh
npm run test
```
